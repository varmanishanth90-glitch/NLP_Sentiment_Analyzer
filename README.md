# NLP Sentiment Analyzer

A machine learning project that performs sentiment analysis on customer reviews using **Python, Scikit-Learn, Flask, and NLTK**.  
This project demonstrates how Natural Language Processing (NLP) can classify text into **Positive 😊** or **Negative 😞** sentiment.

---

## 🚀 Features
- Preprocesses text (cleaning, stemming, stopword removal).
- Converts text into numerical features using **TF-IDF**.
- Trains a **Logistic Regression** model for sentiment classification.
- Simple **Flask web app** with a text box to analyze reviews.
- Displays results instantly in the browser.
